# React Chat

React Chat is a chat app built with React.

Technology used:

Front:
- React
- React Router
- CSS-in-JS

Back:
- Websocket
- MongoDB
- ExpressJS
- RESTful API

 Development:
- Eslint
- Prettier
- Jest
- React Testing Library

## Getting started

The project uses Yarn as package manager.

### Start the project

```bash
yarn start
```

Add `:dev` to the command to start the project in dev mode.

### Build the project

```bash
yarn build
```

Add `:dev` to the command to build the project in dev mode.

### Test the project

```bash
yarn test
```
